# AI_Hair_Style

Бесплатный конфеденциальный редактор ваших фото

Можно легко подобрать причёску в пару кликов, посмотреть как вы будете выглядеть с длинными или короткими или вообще с волосами другого цвета

Чтобы запустить скачайте файл main.py и запустите его через терминал в папке его расположения через команду python main.py

Как установить: 
Для torch

Елси есть видеокарта pip install torch torchvision --index-url https://download.pytorch.org/whl/cu132
Если нету хорошей видеокарты pip install torch torchvision

Для остального:

pip install diffusers

pip install transformers accelerate

pip install mediapipe

pip install opencv-python

pip install numpy Pillow

pip install gradio
